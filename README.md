# auth-task-vinnovate

## Run Test Cases
1. Open Terminal in backend folder
  
2. run: ``` npm start ```

3. Open another Terminal in backend folder

4. run: ``` npm test ```
  

## Run Whole app
1. Open Terminal in backend folder
  
2. run: ``` npm start ```

3. Open Terminal in frontend folder

4. run: ``` npm start ```
