# auth-task-vinnovate

## Run Test Cases
1. Open Terminal in backend folder.

2. run: ``` npm install ```
  
3. run: ``` npm start ```

4. Open another Terminal in backend folder.

5. run: ``` npm test ```
  

## Run Whole app
1. Open Terminal in backend folder.

2. run: ``` npm install ```
  
3. run: ``` npm start ```

4. Open Terminal in frontend folder.

5. run: ``` npm install ```

6. run: ``` npm start ```
